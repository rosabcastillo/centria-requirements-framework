# Centria AI-Assisted Delivery Framework

A lightweight, Git-based requirements system that gives BAs, PMs, QAs, and Devs a single source of truth — one that Claude Code can read directly.

All requirements, decisions, and test plans live as markdown files in this repo. Jira stays for status tracking and stakeholder visibility. Service repos point here for their specs. Skills in `skills/` guide each role through common tasks — they work in Claude Code or as plain prompts in claude.ai.

See [FRAMEWORK_HANDOFF.md](FRAMEWORK_HANDOFF.md) for the complete guide.
